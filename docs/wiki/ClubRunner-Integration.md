# ClubRunner Integration Guide

This page explains how to take the HTML generated by the Highlights System and successfully integrate it into the Seoul Hanmaum Rotary Club newsletter via ClubRunner.

## 1. Get the HTML
1.  Complete the selection process in the [Admin Dashboard](Admin-Guide).
2.  Click **"Copy ClubRunner HTML"**. This places the code on your clipboard.

## 2. In ClubRunner
1.  Log in to the **ClubRunner Member Area**.
2.  Navigate to **Communication** > **Email Service**.
3.  Choose **Create New Email** or edit an existing draft.
4.  In the email designer, look for the **"Custom HTML"** widget/block.
5.  Drag the Custom HTML block into your email layout.
6.  Click on the block to open the editor.
7.  **Paste** the copied code into the HTML text area.
8.  Save the block.

## 3. Previewing
> [!IMPORTANT]
> Always use the **"Send Test Email"** feature in ClubRunner to verify how the highlights look on both **Desktop** and **Mobile** devices. 

The generated HTML uses a responsive layout:
*   On **Desktop**, it maintains a professional 600px width.
*   On **Mobile**, images and text will scale to fit the screen width.

## 4. Customizing the Style
If you need to change fonts or colors for a specific month, you can modify the `<style>` block at the top of the pasted HTML code before saving in ClubRunner. However, it is recommended to use the default styles to maintain club brand consistency.
