# Admin Guide: Managing Highlights

The **PR and Marketing Chair** is responsible for curate the submitted highlights and preparing the monthly newsletter content.

## 1. Accessing the Dashboard
*   Log in with an authorized email address (Contact the Tech Committee if you need access).
*   Click on **"Admin Dashboard"** in the top navigation bar.
*   If you don't see this button, your email is not in the `ADMIN_EMAILS` list.

## 2. Dashboard Features

### The Submission List
All recent member submissions are displayed here. You can see:
*   The member's profile photo.
*   Their professional title and company.
*   The **Highlight Title** (new).
*   The **Highlight Body** with formatting (Bold/Links) preserved.

### Selecting Highlights
*   **Click on a card** to select or deselect it. A blue border indicates the member is selected for export.
*   You can select as many members as you wish for a single newsletter.

### Language Selection
*   Use the **Language Dropdown** (Export EN / Export KO) to decide which version of the content you want to copy.
*   This determines the header text ("Member Highlights" vs "회원 하이라이트") and the layout of the generated cards.

## 3. Exporting to ClubRunner
*   Once you have selected the desired members and chosen the language, click **"Copy ClubRunner HTML"**.
*   A confirmation alert will appear: *"ClubRunner HTML copied to clipboard!"*
*   You are now ready to paste this into the ClubRunner email editor.
